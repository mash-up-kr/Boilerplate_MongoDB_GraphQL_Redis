## Node.js MongoDB & GraphQL & Redis Boilerplate

The node.js environments are following.
* Node.js v14.15.4
* MongoDB v4.4.1
* Redis v6.0.8

## How to install

```
git clone https://github.com/mash-up-kr/Boilerplate_MongoDB_GraphQL_Redis.git
cd ./Boilerplate_MongoDB_GraphQL_Redis
npm install
npm run start
```
